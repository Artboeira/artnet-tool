Three explicit tabs: Record, Perform, Configure. Each mode owns the full canvas. Modes are unambiguous — you can only be in one at a time. Clear for new users; designer-oriented structure.

Direction 6 Operator Mode — Show Mode preview

The V2 Show Mode / operator mode concept: locked view, maximum text size, nothing but scene tiles. This is what the venue operator sees on the back-room PC. Designed to be readable from a sticky note and across the room. Designer-configured, operator-operated.
